# Generate gray code patterns

# Instructions

1. Open `thanh_graycode.py`
2. Change the following parameters
```
# Set initial parameters
im_h = 1080
im_w = 1920
# Number of bits
bit_no = 5
```
3. If necessary, make a gif and let it run, it looks cool.
